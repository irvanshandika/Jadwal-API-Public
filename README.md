> # Jadwal-API-Public

Silahkan jika ingin menggunakannya.

CLI:

Clone Repo:

`git clone https://github.com/irvanshandika/Jadwal-API-Public.git`

Install nodemon:

`npm i -g nodemon`

Run the API:

`npm start` or `nodemon index`

The default port: 5000
